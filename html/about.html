About BoomAPI

BoomAPI aims to be the best source of Boom Arena statistics on the Internet. This project was started in order to provide real statistics to help the balance team, however it may also serve to help players find decks, or analyze their gameplay.
The BoomAPI source code, both backend and frontend are open-source on GitHub.

How does it work?

A server running 24/7 pretends to be a game client, and can then collect data from public replays. As a result of many games not being public, the server is unable to collect data from every game.

I've discovered a bug on the website. How do I report it?

Report bugs directly to me through direct messages (silverslither_ on Discord).

I'm a developer. Can I use BoomAPI in my app?

BoomAPI's endpoints are hosted on the api.boomapi.ca subdomain. Documentation will be available soon.
